### 1.star

简单介绍：

首先从font-awesome图标库引入star图标，创建input标签与label标签；

然后把input标签设置为display:none，为label标签添加before伪类，写为黑色star的样式；

再为label标签添加after伪类，写为蓝色star的样式；

最后，添加star被hover，checked时的样式变化。
